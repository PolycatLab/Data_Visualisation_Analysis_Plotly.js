# Data Visualisation and Analysis using Plotly.js

Examples of data visualisation and data analysis using Plotly.js directly on a data repository. The script is directly embeded in the index.html page uploaded at the same time as the data. The data are in a separate .csv file

Examples and screeshots from doi: [10.14469/hpc/8729](doi.org/10.14469/hpc/8729)
